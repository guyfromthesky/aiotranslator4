Spec file is to complile the application with additional configuration.
In this case, it's additonal hook folder, icon and files.

To use spec file, please use command line to run with this syntax:

* path to your pyinstaller (translator environment) * path to spec file.
Example:

"D:\Translator Env\Scripts\pyinstaller.exe" "D:\GitHub\aiotranslator4\spec\Translate Helper.spec"
